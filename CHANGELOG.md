# Changelog

## 0.3.0

This release establishes the cleaned post-refactor baseline for ToolAgents.

Highlights:

- removed the backward-compatibility layer and standardized on the current public API
- stabilized provider settings and core agent/message surfaces
- cleaned up optional dependencies and grouped them by feature area
- retired unsupported legacy tests and generated example artifacts
- aligned the README, docs, and installation guidance with the maintained API
